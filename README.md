# Expense Tracker App

A simple and user-friendly Expense Tracker application built with React and Vite. Add income and expense transactions, view summaries, and monitor spending with a category-based chart.

## Features

- Add income and expense transactions
- View total income, total expenses, and current balance
- Category-based expense doughnut chart
- Search and delete transactions
- Data stored locally in browser

## Run locally

```bash
npm install
npm run dev
```

Open http://localhost:5173/ in your browser.

## Build

```bash
npm run build
npm run preview
```
